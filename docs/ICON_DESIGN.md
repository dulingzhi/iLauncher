# iLauncher 图标设计

## 设计理念

iLauncher 的图标融合了**搜索**和**快速启动**两大核心功能：

### 视觉元素

1. **渐变蓝色圆形背景** 🔵
   - 深蓝色 (#4299E1) 到青色 (#38B2AC) 的渐变
   - 象征现代化、科技感
   - 圆形代表搜索框的外观

2. **白色放大镜图标** 🔍
   - 核心搜索功能的视觉表达
   - 清晰的轮廓，易于识别
   - 位于图标中心略偏左位置

3. **金黄色小火箭** 🚀
   - 右上角的点缀元素
   - 象征"快速启动"功能
   - 红色尾焰增加动感

### 颜色方案

- **主色调**: 蓝色系（#4299E1, #38B2AC）
- **强调色**: 白色（#FFFFFF）- 图标主体
- **点缀色**: 金黄色（#FFDF00）- 火箭
- **细节色**: 红色（#FF6B6B）- 火箭尾焰

## 技术规格

### 文件格式

- **Windows**: `icon.ico` (多尺寸：16x16 到 256x256)
- **macOS**: `icon.icns` (包含 16x16 到 512x512@2x)
- **Linux**: `icon.png` (512x512)
- **通用**: 各种 PNG 尺寸（30px 到 310px）

### 生成方式

图标通过 Python + Pillow 自动生成：

```bash
# 生成所有 PNG 和 ICO
python scripts/generate_icon.py

# 生成 macOS ICNS
python scripts/generate_icns.py
```

### 文件清单

```
src-tauri/icons/
├── icon.png          # 主图标 (512x512)
├── icon.ico          # Windows 图标
├── icon.icns         # macOS 图标
├── 32x32.png         # Tauri 配置所需
├── 128x128.png       # Tauri 配置所需
├── 128x128@2x.png    # 高清版本
└── Square*.png       # Windows Store 各尺寸
```

## 设计特点

✅ **简洁现代** - 扁平化设计，符合现代 UI 趋势
✅ **高辨识度** - 独特的放大镜+火箭组合
✅ **多平台适配** - 在各种背景和尺寸下都清晰可见
✅ **功能直观** - 一眼就能看出是搜索启动工具
